# AI & Data Science Portfolio Website

A modern and interactive portfolio website built using Streamlit to showcase my skills, projects, certifications, and experience in Data Science, Machine Learning, Artificial Intelligence, and Software Development.

### Features

* Professional responsive design with dark theme
* About Me section with career objective
* Technical skills visualization with progress bars
* Project showcase with GitHub links
* Certifications gallery
* Resume download functionality
* Contact form for recruiters and hiring managers
* Interactive dashboards and data visualizations
* Visitor counter and social media integration

### Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Plotly
* Matplotlib
* Scikit-learn
* HTML/CSS

### Purpose

This portfolio serves as a central platform to demonstrate my technical expertise, academic background, practical projects, and passion for Data Science, Machine Learning, and Artificial Intelligence. It is designed to help recruiters, hiring managers, and collaborators easily explore my work and professional profile.
